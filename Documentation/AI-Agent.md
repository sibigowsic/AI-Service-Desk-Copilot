# AI Agent Design

## Agent

Service Desk Agent

---

## Responsibilities

The agent performs enterprise IT support tasks through natural language conversations.

---

## Available Tools

### Search Knowledge

Purpose

Searches ServiceNow Knowledge Base using AI Search Retrieval.

---

### Create Incident

Purpose

Creates new ServiceNow Incident records.

---

### Get Incident Status

Purpose

Retrieves incident information using the Incident Number.

---

### Update Incident

Purpose

Adds work notes to existing incidents.

---

### Close Incident

Purpose

Marks incidents as closed after successful resolution.

---

## Design Principles

- Natural Language First
- Tool-Based Architecture
- Enterprise ITSM Workflow
- Knowledge Before Ticket
- Human Friendly Responses

---
